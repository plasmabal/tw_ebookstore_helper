# Taiwan Ebookstore Helper (台灣電子書商小幫手)

這是一個專為台灣電子書讀者開發的 Chrome 擴充功能，旨在優化各家電子書平台的購書體驗。目前主要針對 Readmoo 讀墨提供便利的價格計算與領書額度建議功能。

[![Chrome Web Store](https://img.shields.io/chrome-web-store/v/mmemokbgpmfkapbcaddcongejnjjepba.svg)](https://chromewebstore.google.com/detail/taiwan-ebookstore-helper/mmemokbgpmfkapbcaddcongejnjjepba)
[👉 前往 Chrome 線上應用程式商店安裝](https://chromewebstore.google.com/detail/taiwan-ebookstore-helper/mmemokbgpmfkapbcaddcongejnjjepba)

## 專案目的

在台灣各電子書平台購書時，讀者常需要計算折扣後的實際金額，或是評估該使用現金購買還是使用領書額度（例如 Readmoo 的領書本數）。本工具自動擷取網頁上的價格資訊，並即時計算對應的數據，幫助讀者做出最划算的購書決策。

## 目前功能

### 1. Readmoo 讀墨輔助
*   **自動偵測價格**：在書籍詳細頁面自動辨識「電子書售價」或「電子書特價」。
*   **最佳購買法互動組件**：
    *   **智慧最優解**：自動計算「75折」、「8折」、「-50元折價券」以及「領書點數」四種方案的實際成本。
    *   **極簡視覺**：僅以綠色長橢圓按鈕顯示最划算的選項標籤（如 `75折 v` 或 `領書額度 1 點 v`）。
    *   **詳細資訊選單**：點擊按鈕可展開下拉選單，查看所有折扣方案的具體試算金額。
*   **待購清單備註功能**：
    *   **即時捕捉**：在書籍詳情頁點擊「加入待購清單」時，自動彈出備註輸入框，記錄想買的原因。
    *   **清單整合顯示**：在待購清單頁面 (`#wishlist`) 直接顯示書籍備註，並忠實呈現多行換行格式。
    *   **自動同步清理**：進入待購清單頁面時，系統會自動比對並移除已購買或已不在清單中的書籍備註，確保本機儲存空間永不冗餘。
    *   **便捷編輯**：在清單中直接點擊備註內容即可即時修改。

### 2. 名單過濾與突顯
*   **全站覆蓋**：功能已擴展至 Readmoo 首頁、出版社頁面以及各類書籍列表（Grid/List 模式）。
*   **自訂名單**：
    *   **🚫 黑名單 (過濾)**：符合名單的出版社或作者會被標上紅色 `X`，書名變淡並加上刪除線。
    *   **✅ 優良名單 (突顯)**：符合名單的出版社或作者會以「深綠色橢圓色塊」突顯，方便快速辨識優質內容。
*   **備註功能**：每條名單皆支援「多行備註」，可詳細紀錄過濾或突顯的原因，資料完全儲存在本機。
*   **視覺標記**：
    *   符合名單的出版社或作者名字會被標上紅色 `X` 或綠色底塊（包含出版社頁面的大標題）。
    *   符合名單的書籍名稱在所有列表中都會自動變淡並加上刪除線，方便快速過濾。
*   **專業管理中心 (Management Center)**：點擊工具列圖示「⚙️ 設定」即可開啟。
    *   **側邊欄導覽**：採用現代化的設計，將黑名單、優良名單與系統工具分類管理。
    *   **即時計數**：各分類旁設有計數泡泡，一目瞭然資料規模。
    *   **🛠️ 資料備份與還原**：支援將全系統資料（名單、備註）匯出為 JSON 檔案，或從檔案還原，並支援直覺的拖拉式匯入。
    *   **🎨 全新視覺識別**：採用扁平化專業向量圖示，在任何瀏覽器主題下皆能清晰呈現。
    *   **寬敞編輯空間**：提供寬闊的主內容區域，方便編輯與閱讀長篇的多行備註。
    *   **快速新增**：支援按 Enter 鍵快速新增名單。

## 安裝說明

### 商店版本 (推薦)
直接前往 [Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/taiwan-ebookstore-helper/mmemokbgpmfkapbcaddcongejnjjepba) 點擊「加到 Chrome」。

### 開發者版本
1. 下載此專案原始碼。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的「開發者模式」。
4. 點擊「載入未封裝項目」，選擇此專案目錄。
