# Taiwan Ebookstore Helper (台灣電子書商小幫手)

這是一個專為台灣電子書讀者開發的 Chrome 擴充功能，旨在優化各家電子書平台的購書體驗。目前支援 Readmoo 讀墨、博客來、Kobo TW，提供價格試算、名單標記等便利功能。

[![Chrome Web Store](https://img.shields.io/chrome-web-store/v/mmemokbgpmfkapbcaddcongejnjjepba.svg)](https://chromewebstore.google.com/detail/taiwan-ebookstore-helper/mmemokbgpmfkapbcaddcongejnjjepba)
[👉 前往 Chrome 線上應用程式商店安裝](https://chromewebstore.google.com/detail/taiwan-ebookstore-helper/mmemokbgpmfkapbcaddcongejnjjepba)

## 專案目的

在台灣各電子書平台購書時，讀者常需要計算折扣後的實際金額，或是評估該使用現金購買還是使用領書額度（例如 Readmoo 的領書本數）。本工具自動擷取網頁上的價格資訊，並即時計算對應的數據，幫助讀者做出最划算的購書決策。

> **注意**：extension 的最優價格計算僅供參考，實際折扣請以書商系統為主。

## 目前功能

### 1. Readmoo 讀墨輔助

#### 1.1 價格試算小幫手

*   **適用頁面**：書籍詳情頁、活動頁宣傳卡片、每日特惠書列表、活動延伸閱讀列表、待購清單。
*   **四種方案自動試算**：
    *   **75折**：售價 × 0.75
    *   **8折**：售價 × 0.80
    *   **-50元折價券**：售價 − 50
    *   **領書額度**：依 Readmoo 領書點數換算（每 $250 以內 1 點，每點約 $167）
*   **完整下拉選單**（書籍詳情頁、活動頁宣傳卡片）：點擊主按鈕展開，最佳方案標示「(最佳)」，主按鈕常態顯示最划算選項。
*   **輕量綠色標籤**（特惠列表、待購清單）：以 `↳ {最佳選項}` 格式顯示於價格下方。
*   **智慧過濾**：若「領書額度兌換」按鈕為停用狀態（如套書等不適用品項），自動略過領書額度選項。

#### 1.2 待購清單備註與標籤

*   **加入時彈出備註浮窗**：點擊「加入待購清單」後，自動彈出備註輸入框，可記錄備忘及自訂標籤（Tag）。
*   **Tag autocomplete**：輸入時自動提示常用標籤；書籍移出清單後標籤自動保留為 Template，供日後重複使用。
*   **Tag 篩選狀態條**：待購清單頁上方顯示所有標籤，支援多標籤 AND 交集篩選，無符合時顯示提示訊息。
*   **清單顯示與行內編輯**：在待購清單頁直接顯示備註與標籤，點擊容器即可即時修改。
*   **移除時自動清理**：點擊移除書籍，或每次進入清單頁，自動清除已不在清單中的孤立備註與標籤資料。

### 2. 名單過濾與突顯

**適用平台**：Readmoo 讀墨、博客來、Kobo TW

*   **自訂名單**：
    *   **🚫 黑名單（過濾）**：符合名單的出版社或作者，書名變淡並加刪除線，標上紅色 X。
    *   **✅ 優良名單（突顯）**：符合名單的出版社或作者，以深綠色橢圓色塊突顯，方便快速辨識。
    *   黑名單優先於優良名單，同時符合時以黑名單為準。
*   **適用頁面**：書籍詳情頁、作者／出版社頁（Readmoo）、各類列表及搜尋頁。
*   **備註功能**：每條名單支援多行備註，可記錄過濾或突顯的原因，資料完全儲存於本機。

### 3. 管理中心

點擊工具列圖示 → 「開啟設定」，或直接開啟 `management.html`。

*   **黑白名單 CRUD**：新增、顯示、行內編輯、刪除四個清單（出版社黑、作者黑、出版社白、作者白）；支援按 Enter 快速新增，不允許重複名稱。
*   **名單標籤管理**：統計所有名單條目使用的 tag 及次數，支援重命名（批次更新）與刪除。
*   **待購清單標籤管理**：統計所有待購書籍使用的 tag，合併顯示 Template tag，支援重命名與刪除。
*   **資料備份與還原**：匯出全系統資料為 JSON 檔，或從 JSON 檔（支援拖曳匯入）還原；匯入前顯示確認對話框。
*   **現代化側邊欄導覽**：各分類旁設有即時計數泡泡，快速掌握資料規模。

### 4. 跨裝置同步

資料儲存於 `chrome.storage.sync`，登入同一 Chrome 帳號的多台裝置自動同步。Edge 等其他 Chromium 瀏覽器同樣支援，但各瀏覽器帳號體系獨立，無法跨瀏覽器同步。

## 安裝說明

### 商店版本 (推薦)
直接前往 [Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/taiwan-ebookstore-helper/mmemokbgpmfkapbcaddcongejnjjepba) 點擊「加到 Chrome」。

### 開發者版本
1. 下載此專案原始碼。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的「開發者模式」。
4. 點擊「載入未封裝項目」，選擇此專案目錄。
