# Taiwan Ebookstore Helper (台灣電子書商小幫手)

這是一個專為台灣電子書讀者開發的 Chrome 擴充功能，旨在優化各家電子書平台的購書體驗。目前主要針對 Readmoo 讀墨提供便利的價格計算與領書額度建議功能。

## 專案目的

在台灣各電子書平台購書時，讀者常需要計算折扣後的實際金額，或是評估該使用現金購買還是使用領書額度（例如 Readmoo 的領書本數）。本工具自動擷取網頁上的價格資訊，並即時計算對應的數據，幫助讀者做出最划算的購書決策。

## 目前功能

### 1. Readmoo 讀墨輔助
*   **自動偵測價格**：在書籍詳細頁面自動辨識「電子書售價」或「電子書特價」。
*   **折扣與領書雙顯示**：同時顯示「75 折預估價」與「所需領書點數」，並在較划算的選項前加上 `✅` 標記。
*   **介面整合**：例如顯示為 `(✅ 75折: 188 | 領書 1 點)` 或 `(75折: 322 | ✅ 領書 2 點)`，直接插入網頁價格區塊下方。

### 2. 黑名單過濾
*   **全站覆蓋**：功能已擴展至 Readmoo 首頁、出版社頁面以及各類書籍列表（Grid/List 模式）。
*   **自訂名單**：使用者可自訂「出版社」與「作者/譯者」黑名單，資料完全儲存在本機。
*   **視覺標記**：
    *   符合黑名單的出版社或作者名字會被標上紅色 `X`（包含出版社頁面的大標題）。
    *   符合名單的書籍名稱在所有列表中都會自動變淡並加上刪除線，方便快速過濾。
*   **便捷管理**：點擊工具列圖示可快速開啟黑名單管理頁面，支援按 Enter 鍵快速新增。

## 安裝說明

1. 下載此專案原始碼。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的「開發者模式」。
4. 點擊「載入未封裝項目」，選擇此專案目錄。
