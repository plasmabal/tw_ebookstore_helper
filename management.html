🚫 出版社黑名單

符合此名單的出版社，書籍名稱將會變淡並加上刪除線。

    🚫 作者/譯者黑名單

    符合此名單的作者，書籍名稱將會變淡並加上刪除線。

      ✅ 出版社優良名單

      符合此名單的出版社，會以深綠色橢圓背景突顯。

        ✅ 作者/譯者優良名單

        符合此名單的作者，會以深綠色橢圓背景突顯。

          🏷️ 名單標籤

          管理黑名單與優良名單中使用的標籤。可以重命名或刪除標籤，操作會套用至所有使用該標籤的條目。

          🏷️ 待購清單標籤

          管理待購清單中書籍的標籤。

          📖 讀墨閱讀器設定

          調整讀墨電子書閱讀器的行為。

          在試讀頁面自動關閉「喜歡這本書嗎？加入購物車吧！」和「該讓眼睛休息一下囉！」對話框，不再打擾閱讀。

          🛠️ 資料備份與還原

          在此匯出您的所有設定與名單，或從 JSON 檔案還原。這對於在不同環境間遷移資料非常有用。

          📤 備份資料

          將目前所有的黑名單、優良名單與待購備註匯出為 JSON 檔案。

          📥 還原資料

          從備份檔案還原資料。您可以點擊下方按鈕選取檔案，或直接將 JSON 檔案拖拉至此處。
